# ❌⭕ Tic Tac Toe

**`Jogo da velha interativo com modo multiplayer e inteligência artificial`**

Projeto desenvolvido com foco em lógica de programação, interatividade e experiência do usuário, implementando um jogo da velha completo com **modo contra jogador e contra CPU**.

---

## 🚀 Sobre o Projeto

O **Tic Tac Toe** é uma aplicação web que simula o clássico jogo da velha, com interface moderna e funcionalidades que vão além do básico.

O projeto foi desenvolvido para praticar conceitos como:

➽ Manipulação do DOM  
➽ Lógica de jogo  
➽ Estruturação de código  
➽ Algoritmos de decisão (IA)  

---

## 🎮 Funcionalidades

➽ Modo **2 jogadores (PvP)**  
➽ Modo **vs CPU (inteligência artificial)**  
➽ Sistema de pontuação (X, O e empates)  
➽ Destaque visual para o vencedor  
➽ Animações suaves nas jogadas  
➽ Botão de reinício de partida  
➽ Interface responsiva e moderna  

---

## 🤖 Inteligência Artificial

O modo contra CPU utiliza o algoritmo **Minimax**, que analisa todas as possibilidades do jogo para tomar a melhor decisão possível.

✔️ A CPU nunca perde  
✔️ Sempre escolhe a jogada mais estratégica  
✔️ Simula múltiplos cenários antes de jogar

## 📊 Lógica do Jogo

O sistema funciona com base em:

➽ Controle de estado do tabuleiro (array)  
➽ Verificação de combinações vencedoras  
➽ Alternância de jogadores (X e O)  
➽ Detecção de empate  
➽ Atualização dinâmica da interface  

---

## 🎯 Diferenciais

➽ Implementação de IA com **Minimax**  
➽ Interface moderna com animações  
➽ Código organizado e reutilizável
